# Simulação de um Sistema Bancário Simples

Este é um projeto de simulação de um sistema bancário simples em Python. O programa permite que os usuários façam depósitos, saques e verifiquem seus extratos bancários.

## Enunciado do Desafio

"Neste projeto, você terá que criar um Sistema Bancário em Python. O objetivo é implementar três operações essenciais: depósito, saque e extrato. O sistema será desenvolvido para um banco que busca monetizar suas operações. Durante o desafio, você terá a chance de aplicar seus conhecimentos em programação Python e criar um sistema funcional que simule as operações bancárias. Prepare-se para aprimorar suas habilidades e demonstrar sua capacidade de desenvolver soluções práticas e eficientes"

## Funcionalidades

- **Depósito:** Os usuários podem fazer depósitos em suas contas bancárias, especificando o valor desejado.

- **Saque:** Os usuários podem efetuar saques, desde que não ultrapassem o limite diário e não excedam o saldo disponível.

- **Extrato Bancário:** Os usuários podem visualizar o extrato de suas transações, incluindo depósitos e saques anteriores.

- **Sair:** Os usuários podem sair do sistema bancário a qualquer momento.

## Tecnologias Utilizadas

- **Linguagem de Programação:** Python

## Uso

1. Execute o programa em um ambiente Python.
2. Siga as instruções na tela para realizar operações bancárias, como depósito, saque e verificação do extrato.

## Limitações

- O limite diário para saques é de 3 vezes.
- O limite máximo de saque por operação é de R$500,00.
- Não é possível efetuar depósitos com valores negativos.
